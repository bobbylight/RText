Saving the Current Text Document with a new File Name

You can save the current text document as a web page for easy publishing of your source code on the web by doing any of the following:

  1. Go to the File menu, then select 'Save a copy as Web Page...'. You will be prompted for a file name for the HTML file.
  2. Define a shortcut for this action and use it.
