The 'File Chooser' Options Panel

The 'File Chooser' Options panel allows you to:

  1. Toggle whether or not to display hidden files in the file chooser, and with what color to display them.
  2. Toggle whether to use system icons in the file chooser. If this option is enabled on a system that uses special icons to represent different file types (such as Microsoft Windows), these icons will be used. If this option is disabled, or RText is being run on a system with no system icons, default icons will be used.
  3. Toggle whether the "file name" text field displays an "auto-complete" list of matching filenames as you type.
  4. Toggle whether to use colors to differentiate file types in the file chooser, and what colors to use for each file type.
