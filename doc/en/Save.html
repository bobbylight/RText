Saving the Current Text Document

You can save the currently-active text document in one of three ways:

  1. Click on the Save button on the Toolbar. If the current document has not been previously saved, you will be prompted for a file name.
  2. Go to the File menu, then select Save. If the current document has not been previously saved, you will be prompted for a file name.
  3. Press Ctrl+S (or whatever shortcut you have defined for this action). If the current document has not been preiously saved, you will be prompted for a file name.

See also: Saving all open Text Documents, Saving the Current Text Document with a New File Name
