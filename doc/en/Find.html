Find

There are several ways to search for a word or phrase in a document in RText:

  1. Click on the Find button on the Toolbar.
  2. Go to the Edit menu, then select Find.
  3. Press Ctrl+F (or whatever shortcut you have defined for this action).

The Find Dialog will appear, allowing you to enter the text to search for.

As an alternative, you can enable the QuickSearch toolbar for a quick and easy way to search for plain text.

For a tutorial on searching with regular expressions in RText, please see Using Regular Expressions.
