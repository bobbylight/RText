The Options Dialog

The Options Dialog allows you to change many features in RText. It allows you to:


  • Change the layout and appearance of RText.
  • Change virtually all proeprties of the text editor, including the syntax highlighting color scheme.
  • Manage Macros.
  • Configure properties of the custom file chooser.
  • Change the language and localization used in RText.
  • Configure the file filters used to associate programming languages with files.
  • Configure the Source Browser.
  • Change the shortcuts for various menu actions.
