The QuickSearch Toolbar

The QuickSearch toolbar allows you to search for text without bringing up the Find dialog. When it is visible, pressing Ctrl+F (or whatever shortcut you have defined) will shift focus to the text field on the QuickSearch bar. Type the text you wish to find here; then,

  • Press Enter to find the next match in the doucment.
  • Press Ctrl+Enter to find the previous match.
Its visibility can be toggled via the View menu.
