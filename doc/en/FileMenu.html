The File Menu

The File menu contains menu items to help you open, edit and save text documents. It contains the following menu items:


Action What It Does
New Create a new text document.
Open Open a text document.
Open in new window... Open a text document in a new window.
Close Close the current text document.
Close All Close all open text documents.
Save Save the current text document.
Save As... Save the current text document with a new file name.
Save a Copy as Web Page... Save a copy of the current text document as a web page.
Save All Save all open text documents.
Print Print the current text document.
Print Preview... Shows how the current text document will look printed.
Recent Files A list of the most recently-viewed files for quick reopening.
Exit Exits RText.
