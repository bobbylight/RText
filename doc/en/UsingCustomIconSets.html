Using Custom Icon Sets

By default, RText comes with several different icon sets. You can choose what icon set RText uses via the User Interface panel of the Options dialog.

You can add additional icon sets for RText to use, adding a whole level of customization. Additional icon sets are available for download at RText's project page: http://sourceforge.net/projects/rtext. These icon sets are available in Jar files. To make them usable in RText:

  • Take the icon set Jar file, say "NewIconSet.jar", and place it in <RText-root>/extraIcons, where <RText-root> is the directory in which you have RText installed.
  • Add some XML to the file <RText-root>/extraIcons/ExtraIcons.xml as follows: 
              <group name="<name>"
                path="<icon-path>"
                extension="<extension>"
                jar="<jar>"/>
    where:
    • <name> is the name of the icon set, as it will appear in RText's Options dialog.
    • <path> is the path in the Jar file of the icons. If the icons are in the "root directory" of the Jar, this parameter should be the empty string, "".
    • <extension> is the extension of the icon files. This value should be one of gif, jpg, or png, as these are the image file types Swing understands. If this value is not specified, RText assumes an extension of gif.
    • <jar> is the FULL PATH of the Jar file containing the icon set. In this example, this value would be <RText-root>/extraIcons/NewIconSet.jar.

Then, the next time you start RText, the icon set specified in NewIconSet.jar will be available from the Options dialog!
