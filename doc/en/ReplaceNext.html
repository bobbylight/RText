Replace Next

The Replace Next option is a quick way to repeat the last Replace you performed. There are three ways to repeat a Replace:

  1. Click on the Replace Next button on the Toolbar.
  2. Go to the Edit menu, then select Replace Next.
  3. Press F4 (or whatever shortcut you have defined for this action).

RText will attempt to repeat the last Replace you performed, beginning at the current cursor position. If no Replace has been previously performed, the Replace dialog is displayed.
