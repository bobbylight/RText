Toolbar

The Toolbar is the set of icons just beneath the menu bar. The Toolbar provides quick access to the most commonly-used functions in RText. It can be docked on all four sides of the main window, made floatable, or hidden completely via the View menu.

The following table describes all of the actions you can perform from the Toolbar:


Action What It Does
New Opens a new text document.
Open Displays the Open File dialog.
Save Saves the currently active document.
Save All Saves all open documents.
Close Closes the current text file.
Print Prints the currently active text document.
Print Preview Allows you to preview how a document will look printed.
Cut Removes the selected text from the current document.
Copy Copies the currently selected text and places it in the system buffer.
Paste Inserts the text last copied into the current location in the current document.
Delete Removes the currently selected text from the current document.
Find Searches for text in the current document.
Find Next Repeats last Find.
Replace Replaces text in the current document.
Replace Next Repeats last Replace.
Undo Undoes the last action.
Redo Redoes the last action undone.

What's more, you can configure the toolbar to only display the buttons that you frequently use. To do this, simply right-click on the toolbar to bring up a popup menu. From here, you can select/deselect individual buttons to display.
