Editing Text

RText provides all of the standard text editing features. A list of these features is found below:


Name What It Does
Cut Removes the currently selected text from the current document and places it on the system clipboard.
Copy Copies the currently selected text and places it in the system buffer.
Paste Inserts the text last copied into the current location in the current document.
Delete Removes the currently selected text from the current document.
Select All Selects all text in the document.
Time/Date stamp Inserts a time/datestamp into the current document.
