The 'Syntax Highlighting' Options Panel

The 'Syntax Highlighting' Options panel allows you to:

  1. Change the font, foreground and background colors used for all token types when syntax highlighting.
  2. Toggle bracket matching, and select what colors to use while doing so.
  3. Toggle whether to display whitespace (spaces and tabs) in text documents.
  4. Toggle whether to enable 'smooth text' (aka antialiasing). Note that this will considerably slow down the rendering of text.
  5. Toggle whether to enable fractional fontmetrics. This may make the spacing used by some fonts look better.

The "Restore Defaults" button at the bottom changes all options on this panel back to their default values.
