Copying Text

All of the typical text editor ways of copying text are also found in RText. First, select the text to copy. Then, do any of the following:

  1. Click on the Copy button on the Toolbar.
  2. Go to the Edit menu, then select Copy.
  3. Press Ctrl+C (or whatever shortcut you have defined for this action).
  4. Right-click in the current document, then select Copy from the popup menu.
