The 'File Filters' Options Panel

The 'File Filters' Options panel allows you to:

  1. Associate file extensions (such as "*.cpp") with document types to be syntax highlighted (such as "C++ source files"). When you open a text file with an extension that is specified in this option panel, it is syntax-highlighted according to the "file type" you associated with the extension.

The "Restore Defaults" button at the bottom changes all options on this panel back to their default values.
