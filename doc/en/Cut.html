Cut Text

All of the typical text editor ways of cutting text are also found in RText. First, select the text to cut. Then, do any of the following:

  1. Click on the Cut button on the Toolbar.
  2. Go to the Edit menu, then select Cut.
  3. Press Ctrl+X (or whatever shortcut you have defined for this action).
  4. Right-click in the current document, then select Cut from the popup menu.
